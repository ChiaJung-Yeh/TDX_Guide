Chapter 2 軌道運輸資料

軌道運輸係指臺鐵、高鐵、捷運與輕軌，其中捷運系統包含臺北捷運、桃園機場捷運、臺中捷運、高雄捷運，輕軌系統含括新北捷運（淡海輕軌）與高雄輕軌。後續的函式中皆須設定營運機構之參數（operator=），請參照以下表格輸入正確代碼。

軌道運輸營運機構 代碼
臺鐵 TRA
高鐵 THSR
臺北捷運 TRTC
高雄捷運 KRTC
桃園捷運 TYMC
新北捷運 NTDLRT
臺中捷運 TMRT
高雄輕軌 KLRT

此外本資料之下載必須利用 API 金鑰，故函式中須輸入app_id=與app_key=兩參數，亦即在 1.6.2 API 金鑰申請的 APP ID 與 APP Key
