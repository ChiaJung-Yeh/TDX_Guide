Chapter 5 航空與航運資料

本套件所介接的航空資料目前僅包含國內外的航線班表資料。航運資料則含括港口點位資料、航運航線資料、航線靠港順序資料資料。

5.1 航空班表資料

◎ 資料概述
回傳結果包含航空公司代碼、班次號碼、起降機場代碼、出發與到著時間、各星期營運與否等資料。本函式中可提供國內與國際航線之班表資料，僅須設定domestic=之參數即可。

◎ 函式參數

Air_Schedule(access_token, domestic=TRUE, out=FALSE)
表 5.1: Air_Schedule()函式參數設定表
參數 必選填 功能 參數設定值
access_token= 必填參數 [Access Token] 1.6.3 Access Token 取得方式
domestic= 國內或國際航線 TRUE：回傳國內航空班表
FALSE：回傳國際航空班表
out= 選填參數 匯出資料之路徑 FALSE：不匯出資料至本機 [預設值]
若欲輸出此結果，路徑中必須含有.csv或.txt

◎ 程式碼撰寫範例
回傳國內航空班表資料

# 介接國內航空班表
air_schedule_domestic=Air_Schedule(access_token, domestic=TRUE)
# 查看air_schedule_domestic資料
air_schedule_domestic

5.2 港口點位資料

◎ 資料概述
回傳結果為港口名稱與經緯度。

◎ 函式參數

Ship_Port(access_token, dtype="text", out=FALSE)
表 5.2: Ship_Port()函式參數設定表
參數 必選填 功能 參數設定值
access_token= 必填參數 [Access Token] 1.6.3 Access Token 取得方式
dtype= 選填參數 回傳的資料型態 text：純文字形式，其資料型態屬data.frame [預設值]
sf：地理資料形式，其資料型態屬sf
out= 匯出資料之路徑 FALSE：不匯出資料至本機 [預設值]
若回傳的資料型態為「text」：路徑必須含有.csv或.txt
若回傳的資料型態為「sf」：路徑必須含有.shp

◎ 程式碼撰寫範例
回傳全臺所有港口資料（地理資料）

# 介接全臺港口資料
taiwan_port=Ship_Port(access_token, dtype="sf")
# 查看taiwan_port資料
taiwan_port

5.3 航運航線資料

◎ 資料概述
回傳國內航運之航線班表資料。

◎ 函式參數

Ship_Route(access_token, county, out=FALSE)
表 5.3: Ship_Route()函式參數設定表
參數 必選填 功能 參數設定值
access_token= 必填參數 [Access Token] 1.6.3 Access Token 取得方式
county= 縣市代碼 請參照TDX_County表格，並確保該縣市有港口
out= 選填參數 匯出資料之路徑 FALSE：不匯出資料至本機 [預設值]
若欲輸出此結果，路徑中必須含有.csv或.txt

◎ 程式碼撰寫範例
回傳新北市航運航線資料

# 介接新北市航運航線資料
NWT_shiproute=Ship_Route(access_token, "NewTaipei")
# 查看NWT_shiproute資料
NWT_shiproute

5.5 航空出入境人次資料

◎ 資料概述
本函式介接民航運輸常用統計資料，依據所輸入的年度下載並整理各航線及航空公司之國內與國際（含兩岸）航線班機載客人數等營運相關資料。本函式中可提供國內與國際航線之運量資料，僅須設定domestic=之參數即可。

◎ 函式參數

Air_Patronage(ym, domestic=FALSE, out=FALSE)
表 5.5: Air_Patronage()函式參數設定表
參數 必選填 功能 參數設定值
ym= 必填參數 年度與月份 時間格式為YYYY-MM
domestic= 選填參數 國際或國內線 TRUE:國內線<br>FALSE:國際線（預設值）
out= 匯出資料之路徑 FALSE：不匯出資料至本機 [預設值]
若欲輸出此結果，路徑中必須含有.csv或.txt

介接國內線資料時，將會下載所有該年度的資料；而國際線資料則會依據所輸入的ym=參數介接指定月份資料。而輸出資料結果個欄位名稱意義彙整如表5.6。

表 5.6: Air_Patronage()函式輸出結果欄位名稱
前綴名稱 前綴意義 後綴名稱 後綴意義
Flight_ 班次數 _Total 總數（出境+入境）
Seat_ 總座位數 _Departure 出境
Patronage_ 總人次數 _Arrival 入境
LoadFactor_ 承載率（總人次數/總座位數）

◎ 程式碼撰寫範例
回傳國際航線出入境人次資料

# 介接國際航線出入境人次資料
air_patro=Air_Patronage("2024-12")

# 查看air_patro資料
air_patro
